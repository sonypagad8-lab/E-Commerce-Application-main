# E-Commerce Application

A Java Full Stack E-Commerce Web Application developed using Spring Boot, Hibernate, SQL, HTML, CSS, and JavaScript.

## Features
- User Registration and Login
- Product Management
- Shopping Cart
- Order Management
- Payment Integration
- Admin Dashboard
- Customer Dashboard
- Merchant Dashboard

## Technologies Used
- Java
- Spring Boot
- Hibernate
- Maven
- MySQL
- HTML
- CSS
- JavaScript
- Thymeleaf

## Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- Entity Layer

## How to Run
1. Clone the repository
2. Configure MySQL database
3. Update application.properties
4. Run EcommerceApplication.java
5. Open browser and access the application
